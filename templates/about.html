About Time Clock

The TimeClock User Interface is a mobile ClockIn / ClockOut application, aimed to replace the need for hand written time cards for job site Professionals.

The TimeClock Admin Interface allows HR / Payroll Professionals to retrive ClockIn / ClockOut information at anytime in order to view and process the work hours and payroll of Individuals.

Known Tradeoff: It is recognised that refreshing the User Interface will create an additional entry. This tradeoff exist because the application cannot recognised the ligitamacy of the entry and therefore accepts the entry for Admin level post processing.
